# Evaluacion5Movile
Creacion de un proyecto de comida rapida

#Pasos a seguir:

Para empezar a trabajar,solo se necesitará clonar el repositorio:
git clone https://github.com/TuUsuario/Evaluacion5Movile.git

1-Crear una nueva rama para sus cambios:
git checkout -b nombre-de-la-rama-para-trabajar

2-Hacer cambios, confirmarlos y subirlos:
git add .
git commit -m "Descripción de los cambios"
git push origin nombre-de-la-rama-para-la-feature
